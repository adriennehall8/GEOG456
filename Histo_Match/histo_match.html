histogram matching images of the port of long beach, ca

Histogram matching is an image processing technique that allows you to alter an image so that it matches a specified histogram. In this application I transformed aerial view images and topoview maps from USGS to match the histogram of another image. This method is useful for clearly identifying landscape changes over time and for creating creative images.
